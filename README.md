# COVID-19 Web Map Application (2020)

This project presents two interactive web maps visualizing COVID-19 data in the United States for the year 2020.

## Maps
- **Choropleth Map (Rates):**
  https://giruml34.github.io/covid-us-2020-maps/map1.html
- **Proportional Symbol Map (Cases):**
  https://giruml34.github.io/covid-us-2020-maps/map2.html

## Description
The choropleth map shows COVID-19 case rates per 1,000 residents by county, while the proportional symbol map displays total confirmed case counts using scaled circles.

## Data Sources
- COVID-19 case data: The New York Times
- Population data: ACS 2018 5-year estimates
- County boundaries: U.S. Census Bureau

## Libraries
- Mapbox GL JS
