## Description
The choropleth map visualizes COVID-19 case rates per 1,000 residents at the county level.  
The proportional symbol map displays total confirmed COVID-19 cases using scaled circles.

## Data Sources
- COVID-19 data: *The New York Times*
- Population data: *ACS 2018 5-year estimates*
- County boundaries: *U.S. Census Bureau*
