# COVID-19 Maps (United States, 2020)

This project visualizes COVID-19 case rates and total cases across U.S. counties in 2020 using Mapbox GL JS.

## Maps
- Choropleth Map (Case Rates): map1.html
- Proportional Symbol Map (Total Cases): map2.html

## Data Sources
- COVID-19 data: New York Times
- Population: ACS 2018 5-year estimates
- County boundaries: U.S. Census Bureau

## Libraries
- Mapbox GL JS
- Google Fonts
